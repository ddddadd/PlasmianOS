Plasmian OS is a brazilian linux distro based on Debian 14 (Forky - Beta), which has KDE Plasma as a default desktop ambient.
# Who created PlasmianOS?
The creator are the members of Team Plasmian, which consists of a 12 year old toddler (me, that is doing this repository) and a random person i found on a discord server that he agreed to help on the development.
# What PlasmianOS focuses on?
Plasmian OS focuses on being user-friendly and easy to install, and has live mode to the user test it out, it even has a calamares installer to make things easier
